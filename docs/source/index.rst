Welcome to Nirschl et al Implementation's documentation!
========================================================

| Documentation for the code written to reproduce the work of the publication,
| "A deep-learning classifier identifies patients with clinical heart failure using whole-slide images of H&E tissue"
| by Nirschl JJ, Janowczyk A, Peyster EG, Frank R, Margulies KB, et al.
| PLOS ONE 13(4): e0192726. https://doi.org/10.1371/journal.pone.0192726
| Please run the Trainer.py file to train the network. The running directory for python is "./nirschl_el_al"
| List of dependencies is available in the file requirements.txt
